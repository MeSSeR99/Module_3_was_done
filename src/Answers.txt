1. Материнская плата - это основа пк, на эту плату устанавливаются все основные компоненты системы.
Процессор - это центральный элемент системы, проводит все вычесления и контролирует все операции.
ОЗУ - это временное хранилище данных текущих операций.
Долгосрочная память - Постоянное храналище информация, на жестких дисках или других устройствах.

2. Основное отличие между стеком и кучей заключается в том, как они управляют своими элементами. Стек работает по принципу "последним пришел - первым вышел" (LIFO), то есть последний добавленный элемент будет первым удаленным. Куча, напротив, работает по принципу "первым пришел - первым вышел" (FIFO), то есть первый добавленный элемент будет первым удаленным.

3. Мы изучили несколько типов данных:
- Целые числа (int, short, long)
- Вещественные числа (float, double)
- Логические значения (boolean)
- Строки (String)
4. Примитивные типы данных имеют свои собственные значения, в то время как ссылочные типы данных представляют собой ссылки на объекты. 

5. Переменная, это какое либо значение которое мы задаем, для использования его в различных процессах.

6. Оператор ! (не) - проверяет значение с условием "не", если значение с этим оператором не верно, тогда значение True
Оператор & (и) - если оба значения верны, тогда значение True 
Оператор | (или) - если хотя бы одно значение верно, тогда True
